# e-plantShopping

## Project Overview
**e-plantShopping** is a React-based web application designed to provide users with a seamless online plant shopping experience. The application includes a landing page, a plants catalog page, and a cart section. Users can browse plants, add them to their cart, and view the total items in the cart.

### Features
- **Landing Page:** Attractive UI with a "Get Started" button to navigate to the plant catalog.
- **Navbar:** Contains links to Paradise Nursery (home), Plants (catalog), and Cart icon (cart section).
- **Plants Page:** Displays plant cards with images, names, prices, and "Add to Cart" functionality.
- **Cart Section:** Allows users to view selected items and manage their cart.

### Technologies Used
- React
- JavaScript
- CSS
- Vite
